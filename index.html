Surgimento da WEB

(Voltar)

A Internet é um meio de comunicação muito utilizado, principalmente nos tempos atuais. Além de facilitar na comunicação e interação social, facilita também o dia-a-dia das pessoas, por exemplo, a utilizando dos e-commerce e afins.

Berners-Lee foi o criador da WEB, mais precisamente do sistema WWW (World Wide Web), a rede de alcance mundial, que interligada vários lugares do mundo simultaneamente. Criou o primeiro web site utilizando HTML (Linguagem de Marcação de Hipertexto) em 1991.

Como se desenvolve toda essa comunicação na Internet? Por meio da interação de Client x Servers. Client é quem inicia a comunicação com o Server. Por exemplo, os navegadores são os clients, são quem inicia a comunicação com o Servs. Que vai consumir informação de algum lugar na Web. Os Servs, os servidores, já demandam um outro comportamento, irá devolver uma resposta ao client. Irá usar a conexão com a Internet para fazer a comunicação e processamento de arquivos, documentos, áudio e etc.

Como funciona o HTML

(Voltar)

Continuando a falar sobre a criação do Tim Berners-Lee, O HTML (HyperText Markup Language) é uma Linguagem de Marcação de Hipertexto.

É atráves desta linguagem que se estrutura websites. O HTML tem a missão de compor as informações das páginas na web. O HTML, é como muitos desenvolvedores falam: é o esqueleto de um site. Pois é no HTML que vão ser colocados as informações, os destaques, todos os elementos necessários para compor o esqueleto do site. Mas, sem a estilização.

O HTML consiste em duas partes: head que é a cabeça e body que é o corpo. Compondo assim, a estrutura básica do HTML.

Atráves deste documento, há por de trás uma série de tags - que são os elementos que compõe a construção do site.

HTML não é uma linguagem de programação. É um arquivo de marcação. Onde os textos são delimitados por nomes que o navegador consegue interpretar - as chamadas tags. E cada nome destes delimitadores, tem uma função e comportamento específico.

O que são as TAGS no HTML?

(Voltar)

As tags no HTML são elementos que tem função e comportamento específico para formatar o conteúdo do documento HTML. Cada TAG tem um nome ou letra e uma função especifica.

Por exemplo, toda tag HTML possui os sinais: (<) (>). Cada tag tem sua característica própria que irá indicar como o navegador irá ler.

Tem-se atributos, ou seja, propriedades que cada tem pode ter. Existe as genéricas, por exemplo "id=" (é uma tag identificadora de elemento), "style=" (coloca comando CSS direto no elemento); como também existe as específicas "input=" (vai fazer campo texto, muito utilizado em formulários de cadastro).
